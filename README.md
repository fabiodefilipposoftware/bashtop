# bashtop
a bash script to transform terminal in a sort of desktop

## DIPENDENCIES
1. tmux
2. ranger file manager
3. xstarter

## INSTALLATION
1. UPDATE AND INSTALL TMUX AND RANGER FILE MANAGER
sudo -s
apt update && apt install tmux ranger
2. DOWNLOAD AND INSTALL XSTARTER FROM GITHUB REPOSITORY
https://github.com/lchsk/xstarter/blob/master/INSTALL.md#deb-package
